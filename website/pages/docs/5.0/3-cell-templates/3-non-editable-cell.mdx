The `NonEditableCell` template is used to display non-editable text values within a cell.

#### Implementation

 - Open `NonEditableCell` implementation implementation on [GitHub](https://github.com/silevis/reactgrid/blob/v5-dev/reactgrid/lib/cellTemplates/NonEditableCell.tsx)

#### Interface declaration 

```ts
interface NonEditableCellProps {
  value?: string;
  style?: React.CSSProperties;
}
```

| Property name  | Type                    | Description                                |
| -------------- | ----------------------- | ------------------------------------------ |
| `value?`       | `string`                | The text value of the cell.                |
| `style?`       | `React.CSSProperties`   | CSS properties to apply to the cell. |
