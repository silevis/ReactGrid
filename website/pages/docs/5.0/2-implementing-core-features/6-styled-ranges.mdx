#### About styled ranges in ReactGrid

In addition to directly passing styles to individual cells, you can also use the `styleRanges` prop, which allows you to ostilise a specific area of cells in a table.

```ts
styledRanges?: StyledRange[];

type StyledRange = {
  styles: React.CSSProperties;
  range: Range;
};

type Range = {
  start: {
    rowIndex: number;
    columnIndex: number;
  };
  end: {
    rowIndex: number;
    columnIndex: number;
  };
};
```
